## Reporting a Vulnerability

If you find a security vulnerability in our project, please **do not open a public Issue**. Use confidential communication channels to prevent attackers from learning about the vulnerability before we can fix it.

1. Send a description of the vulnerability to our email: `r0wner@proton.me`.
2. Please attach steps to reproduce the issue (PoC) or a code example.

### What to expect next:
* **Initial response:** We will acknowledge receipt of your email within `24/48` hours.
* **Analysis:** We will investigate the vulnerability and let you know whether it is accepted or declined.
* **Fix:** If the issue is confirmed, we will release a security patch. Until then, we ask you to follow Responsible Disclosure principles and refrain from publishing vulnerability details online.
