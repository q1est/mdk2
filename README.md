## Описание
**MDK2** — это учебный проект, целью которого является: 
1. Научиться работать в команде
2. Изучить новые технологии

Стек проекта: frontend - HTML, CSS, JS.  Backend - Go. db - Postgres. Хостинг - Hugging Face. Back обернут в docker контейнер. Хостинг db - supabase.

## Возможности

- **Интерактивное меню** - просмотр доступных блюд с описаниями и изображениями
- **Система заказов** - добавление блюд в корзину
- **Адаптивный дизайн** - удобство использования на всех устройствах
- **Современный интерфейс** - стильное оформление и плавные переходы
- **REST API** - мощный бэкенд на Go для управления заказами
- **Docker поддержка** - легкое развертывание с Docker и Docker Compose
- **Log** сбор логов loki, просмотр Grafana

Технологии
Frontend
- **HTML5** - семантическая разметка
- **CSS3** - современное оформление с адаптивностью
- **JavaScript** - интерактивность и логика приложения
- **Vanilla JS** - без зависимостей от фреймворков
Backend
- **Go** - высокопроизводительный язык программирования
- **REST API** - архитектура для работы с данными
- **Docker** - контейнеризация приложения
- **LOKI**
- **Grafana**


Файлы проекта
Frontend
`index.html`
Главная страница приложения с основным контентом и структурой.
`menu.html`
Страница меню с каталогом всех доступных блюд и напитков.
`script.js`
Содержит всю логику приложения:
- Обработка кликов
- Управление корзиной заказов
- Взаимодействие с пользователем
- Динамические изменения интерфейса
`styles.css`
Стилизация всех элементов:
- Макет и сетка
- Цветовая схема
- Анимации и переходы
- Адаптивные стили для мобильных устройств
Backend
`back/main.go`
Инициализация всего приложения

`back/handle/handle.go`
HTTP обработчики (handlers):
- Обработка запросов к меню
- Отправка в db
- Управление заказами
- API endpoints для фронтенда

`back/models/models.go`
Структуры данных:
- Структура блюда (Dish)
- Cтруктура заказа (Order)
- Структура Loki
`back/db/db.go`
- Работа с базой данных:
- Создание таблиц, путем выполнения sql-запроса
`back/db/postgres.go`
- Выполнение подключения к db
- Проверка работоспоспособности

`back/logs/logs.go`
- Функции логирования работоспоспособности приложения и логирования запросов
- Отправка данных в grafana cloud 
`back/tg/tg.go
- Функция отправки заказа (orderhandler) в телеграм (временно не работает, ограничение РКН)

`back/Dockerfile`
- Docker конфигурация для бэкенда:
- Сборка Go приложения
- Настройка контейнера
- Экспозиция портов

`back/docker-compose.yml`
-Docker Compose конфигурация:
- Запуск бэкенда
- Запуск базы данных
- Организация сетей и хранилищ

***Использование***

Frontend

1. **Просмотр меню:**
   - Откройте `menu.html`
   - Изучите доступные блюда и напитки

2. **Добавление в заказ:**
   - Выберите блюдо, кликнув на него
   - Блюдо добавится в вашу корзину

3. **Оформление заказа:**
   - Перейдите в корзину
   - Проверьте выбранные блюда
   - Подтвердите заказ

**Backend**

Бэкенд предоставляет REST API для работы с:
- Получением списка блюд
- Управлением заказами
- Работой с данными клиентов

API использует JSON для обмена данными между фронтендом и бэкендом.
