# MDK2 - Сервис заказа блюд

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Language](https://img.shields.io/badge/frontend-HTML5%20%7C%20CSS3%20%7C%20JS-blue)
![Language](https://img.shields.io/badge/backend-Go-00ADD8)

## 📖 Описание

**MDK2** — это полнофункциональное веб-приложение для заказа блюд и напитков. Проект состоит из фронтенда на HTML/CSS/JavaScript и бэкенда на Go с поддержкой Docker.

## ✨ Возможности

- 🍽️ **Интерактивное меню** - просмотр доступных блюд с описаниями и изображениями
- 🛒 **Система заказов** - добавление блюд в корзину
- 📱 **Адаптивный дизайн** - удобство использования на всех устройствах
- 🎨 **Современный интерфейс** - стильное оформление и плавные переходы
- ⚙️ **REST API** - мощный бэкенд на Go для управления заказами
- 🐳 **Docker поддержка** - легкое развертывание с Docker и Docker Compose

## 💻 Технологии

### Фронтенд
- **HTML5** - семантическая разметка

- **JavaScript** - интерактивность и логика приложения
- **Vanilla JS** - без зависимостей от фреймворков

### Бэкенд
- **Go** - высокопроизводительный язык программирования

- **Docker** - контейнеризация приложения

## 📝 Файлы проекта

### Фронтенд

#### `index.html`
Главная страница приложения с основным контентом и структурой.

#### `menu.html`
Страница меню с каталогом всех доступных блюд и напитков.

#### `script.js`
Содержит всю логику приложения:
- Обработка кликов
- Управление корзиной заказов
- Взаимодействие с пользователем
- Динамические изменения интерфейса

#### `styles.css`
Стилизация всех элементов:
- Макет и сетка
- Цветовая схема
- Анимации и переходы
- Адаптивные стили для мобильных устройств

### Бэкенд

#### `back/main.go`
Главная точка входа приложения:
- Инициализация сервера
- Настройка маршрутов
- Запуск REST API

#### `back/handle/`
HTTP обработчики (handlers):
- Обработка запросов к меню
- Управление заказами
- API endpoints для фронтенда

#### `back/models/`
Структуры данных:
- Модель блюда (Dish)
- Модель заказа (Order)
- Другие бизнес-сущности

#### `back/db/`
Работа с базой данных:
- Подключение к БД
- Операции создания/чтения/обновления/удаления
- Миграции данных

#### `back/Dockerfile`
Docker конфигурация для бэкенда:
- Сборка Go приложения
- Настройка контейнера
- Экспозиция портов

#### `back/docker-compose.yml`
Docker Compose конфигурация:
- Запуск бэкенда
- Запуск базы данных
- Организация сетей и хранилищ

## 🎯 Использование

### Фронтенд

1. **Просмотр меню:**
   - Откройте `menu.html`
   - Изучите доступные блюда и напитки

2. **Добавление в заказ:**
   - Выберите блюдо, кликнув на него
   - Блюдо добавится в вашу корзину

3. **Оформление заказа:**
   - Перейдите в корзину
   - Проверьте выбранные блюда
   - Подтвердите заказ

### Бэкенд

Бэкенд предоставляет REST API для работы с:
- Получением списка блюд
- Управлением заказами
- Работой с данными клиентов

API использует JSON для обмена данными между фронтенда и бэкенда.
